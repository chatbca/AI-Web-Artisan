# **App Name**: AI Web Artisan

## Core Features:

- Prompt-to-Code Generation: Generate website code (HTML, CSS, JavaScript, or React) from a natural language prompt. The tool decides on the overall code structure.
- Live Website Preview: Display a live, interactive preview of the generated website within the app.
- Built-in Code Editor: Integrate the Monaco Editor for direct code modification within the app.
- Code Export: Allow users to export the generated code as a ZIP file.
- Responsive Design: Ensure generated websites are responsive and adapt to different screen sizes.
- Smart Theme Selection: Based on the user prompt, the tool chooses light mode or dark mode as appropriate for the user.
- View Toggle: Enable toggling between the code editor and the website preview in real time.

## Style Guidelines:

- Primary color: Saturated sky blue (#42A5F5) for a feeling of creativity and open possibility.
- Background color: Light, desaturated blue (#E3F2FD) to ensure legibility.
- Accent color: Violet (#9C27B0) to provide a contrasting call to action.
- Font: 'Inter', a grotesque-style sans-serif font known for its readability, for both headings and body text.
- Use simple, outline-style icons to maintain a clean and modern aesthetic.
- Maintain a clean, well-spaced layout to improve readability and overall user experience.
- Subtle animations, like smooth transitions between the code editor and website preview.
